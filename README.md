# 项目简介

- 系统首页管理
- 用户管理
- 主办方管理
- 策划方管理
- 活动分类管理
- 活动信息管理
- 活动报名管理
- 日程规划管理
- 活动提醒管理
- 方案分类管理

- 活动主办方
